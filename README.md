## 🌟 Highlight Project — AI Receipt Tracker

> **A production-grade AI-powered receipt parsing and spending tracker built end-to-end**

| Feature | Details |
|---------|---------|
| 🧠 **AI Parsing** | GPT-4.1-mini vision + GPT-4o-mini text — reads PDFs and images |
| 🗄️ **Database** | PostgreSQL (Supabase) + SQLAlchemy ORM with migrations |
| ☁️ **Cloud Storage** | AWS S3 for receipt file storage with presigned URLs |
| 🔁 **Retry Logic** | Tenacity retry wrapper for rate limits & API timeouts |
| 🧹 **Smart Categorization** | 20+ categories with sub-categories + category memory from history |
| 🔍 **Duplicate Detection** | SHA-256 file hashing to prevent double uploads |
| 📊 **Analytics** | Plotly dashboards — spend by category, store, month + drill-down |
| 🔐 **Auth** | Password-protected with Streamlit session state |
| 💰 **Cost Tracking** | Per-call token logging with monthly API cost monitoring |
| 🧾 **Multi-store Support** | Handles Costco, Target, Walmart — store-specific tax & discount logic |

**Tech:** Python • Streamlit • OpenAI API • PostgreSQL • SQLAlchemy • AWS S3 • Plotly • pandas • boto3
